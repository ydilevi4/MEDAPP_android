# MEDAPP_android

MVP initialization for **MEDAPP**, an Android app to track pill schedules and dose completion.

## What's included in `mvp/init`

- Android app module configured with Kotlin + Jetpack Compose
- Single-screen MVP home (`MainActivity`) with a basic "Get Started" CTA
- Version catalog-based dependency management (`gradle/libs.versions.toml`)

## Next MVP milestones

1. Add onboarding flow for medication setup
2. Create local persistence for medications and dose history
3. Add reminder notifications and missed-dose tracking
